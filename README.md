## Features

- **Categorized Menus & Shopping Cart:** Browse pastries and donuts easily with organized menus and add items to cart.  
- **Quick Ordering:** Place orders in seconds with a smooth checkout process.  
- **Real-Time Order Tracking:** Track order status from **preparing ‚Üí on the way ‚Üí delivered**.  
- **Offline Cart Support:** Users can manage their cart and prepare orders even when offline.  
- **Dashboard & Analytics:** Admin dashboard provides insights on orders, delivery performance, and customer activity.  
- **Optimized Performance:** Smooth and responsive UI with fast loading and offline-first design.

## Getting Started

### Prerequisites

- Flutter and Dart installed on your development machine.  
- [Official Flutter installation guide](https://flutter.dev/docs/get-started/install).  

### Installation

1. Clone the repository:  
```bash
git clone https://github.com/AyaAbdElmoneim158/Carwan_Dough_App
````

2. Navigate to the project directory:

```bash
cd Carwan_Dough_App
```

3. Install dependencies:

```bash
flutter pub get
```

---

## Usage

* Run the app on an emulator or physical device:

```bash
flutter run
```
